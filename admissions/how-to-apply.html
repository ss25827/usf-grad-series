Application Process

Applicants should complete the following steps:

1. Submit Graduate School Online Application.

  • Please note a $30 application fee will be assessed when completing this application
  • Instructions for completing the application can be found here

2. Apply directly to the Online MIS Program by submitting all documents below:

  • Graduate Series Applicant Cover Sheet
  • Resume
  • Statement of Purpose – Include your career goals & how the program will help you accomplish those goals
  • Three Letters of Recommendation – References should be from professors and/or employers
  • College transcripts (copies may be used to expedite the application process)
  • Test scores – GRE or GMAT (copies may be used to expedite the application process)
  • Demonstrate proficiency in English, TOEFL, or IELTS, if applicable

Please submit the above information to:
